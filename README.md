## Installation

sqlite3-arabic-phonetic-fuzzy-trigram is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'sqlite3-arabic-phonetic-fuzzy-trigram'
```
